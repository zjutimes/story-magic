# 📚 Story Magic - AI 儿童绘本生成器

> 输入故事主题，AI 一键生成配有插图和动画的儿童绘本！

## ✨ 功能特点

### 🎨 AI 绘本创作
- 输入故事主题，AI 自动创作完整故事
- 支持中英文双语
- 智能角色设定，同一角色多页面形象一致

### 🎬 插图转动画
- 静态插图一键变成动画视频
- 柔和的镜头移动和角色动画
- 配套音效效果

### 🔊 语音朗读
- 故事文字自动转换为语音
- 儿童有声书风格的声音

### 👤 用户系统
- 注册登录功能
- 创作历史保存

## 🛠️ 技术栈

- **前端**: React + TypeScript + Vite + Tailwind CSS
- **后端**: Express + TypeScript
- **AI 服务**: coze-coding-dev-sdk
  - LLM (豆包模型)
  - 图片生成
  - 视频生成
  - TTS 语音合成
- **数据库**: Supabase Auth

## 🚀 快速开始

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
pnpm dev
```

### 构建生产版本

```bash
pnpm build
pnpm start
```

## 📖 使用流程

1. **注册/登录** - 点击右上角登录按钮
2. **输入主题** - 输入故事主题或选择示例
3. **选择语言** - 中文或英文
4. **生成故事** - 点击"开始创作"
5. **生成插图** - 自动生成配套插图
6. **生成动画** - 将插图变成动画视频
7. **语音朗读** - 让故事开口说话
